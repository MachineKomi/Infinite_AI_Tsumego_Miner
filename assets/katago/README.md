# KataGo Binary

This directory should contain the extracted KataGo binary and its dependencies.

## Instructions

1. **Download:** Get the latest Windows OpenCL release (e.g., `katago-v1.16.4-opencl-windows-x64.zip`) from [KataGo Releases](https://github.com/lightvector/KataGo/releases).
2. **Extract:** Unzip **ALL** contents of the zip file into this directory.
3. **Verify:** You should see `katago.exe` and several `.dll` files here.
